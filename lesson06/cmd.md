# подсказки по командной строке

команда смены директории
```sh
cd c:\folder_name
```

команда отображения текущей директории
```sh
pwd
```
листинг текущей директории
windows:
```sh
dir
```
linex, MacOs:
```sh
ls
```


Удаление файла в Windows 
```sh
del <filemane>
```
в Linex, MacOs:
```sh
rm <filename>
```
